Project Purple Phantom AI

Overview:
- Streamlit web UI with VR HUD styling in `app.py`.
- Kivy mobile UI in `mobile_app.py` (uses remote API if `REMOTE_API_URL` set).
- Flask API in `api_server.py` to host the model endpoint (`/api/respond`).

FNF Codespace:
- See `README-FNF.md` and the `.devcontainer` folder for a Codespaces devcontainer that provides Haxe/Lime/OpenFL and a noVNC desktop.
